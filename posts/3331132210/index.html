[Python] json.dumpsзҡ„еҸӮж•°:ensure_ascii

еҸ‘иЎЁдәҺ 2021-06-10 жӣҙж–°дәҺ 2024-04-29 йҳ…иҜ»ж¬Ўж•°пјҡ
JSONдёӯиЎЁзӨәйқһASCIIеӯ—з¬Ұзҡ„дёӨз§Қж–№ејҸ

иө·жәҗ

иҝҷйғЁеҲҶж— е…іPythonпјҢеҸӘжҳҜи®©жҲ‘жғіиө·дәҶPythonзҡ„json.dumps()иҖҢе·ІгҖӮ

д»ҠеӨ©пјҢжҲ‘еңЁзңӢJSON Linesд»Ӣз»Қзҡ„ж—¶еҖҷзңӢеҲ°дәҶиҝҷд№ҲдёҖеҸҘиҜқпјҡ

JSON allows encoding Unicode strings with only ASCII escape sequences, however those escapes will be hard to read when viewed in a text editor.

еҪ“жҲ‘иҜ»еҲ°иҝҷеҸҘиҜқзҡ„ж—¶еҖҷпјҢжҲ‘иғҪзңӢжҮӮвҖқASCII escape sequencesвҖңжҳҜвҖқASCIIиҪ¬иҜ‘еәҸеҲ—вҖңзҡ„ж„ҸжҖқпјҢдҪҶжҲ‘дёҚжҳҺзҷҪзҡ„жҳҜпјҡJSONдёҚжҳҜиҰҒжұӮз”ЁUTF-8зҡ„д№ҲпјҢдёәд»Җд№ҲиҝҳиҰҒиҪ¬иҜ‘е‘ўпјҹ

еҪ“ж—¶зҡ„жҲ‘пјҢиҝҳдёҚзҹҘйҒ“JSONдёӯиЎЁзӨәйқһASCIIеӯ—з¬Ұзҡ„дёӨз§Қж–№ејҸгҖӮпјҲдёҚеӨӘзЎ®е®ҡиҝҷдёӘиҜҙжі•еҜ№дёҚеҜ№пјү

дёӨз§ҚдёҚеҗҢзҡ„иЎЁзӨәж–№ејҸ

еҰӮжһңдҪ з”ЁиҝҮеҮ ж¬Ўjson.dumps()пјҢдҪ дёҖе®ҡзҹҘйҒ“иҝҷдёӘеҮҪж•°жңүдёҖдёӘensure_asciiеҸӮж•°пјҢиҖҢеҪ“е®ғж”№еҸҳж—¶пјҢиҫ“еҮәзҡ„з»“жһңд№ҹе°ұжңүдёҚеҗҢпјҡ

json дёӯзҡ„ensure_ascii=False пјҲиІҢдјјз”Ёзҡ„жҳҜpython 2пјҢдҪҶдёҚйҮҚиҰҒпјү

еҸҜд»ҘзңӢеҲ°пјҢensure_asciiеҪұе“ҚдәҶеәҸеҲ—еҢ–еҗҺзҡ„жңҖз»ҲеҪўжҖҒгҖӮ

е°Ҫз®Ўж ·еӯҗдёҚеҗҢпјҢдҪҶжҳҜдҪ дјҡеҸ‘зҺ°пјҢдҪҝз”Ёjson.loads()иҜ»еҸ–ж—¶еҚҙе№¶дёҚйңҖиҰҒзү№ж®ҠеӨ„зҗҶпјҢж•ҲжһңжҳҜдёҖж ·зҡ„гҖӮ

иҜ»еҸ–е®һйӘҢ

дёҺдёҠйқўзҡ„еҜ№еә”пјҢеҸіиҫ№зҡ„\u4e2d\u56fdе°ұжҳҜдёӯеӣҪдёӨдёӘеӯ—еңЁUnicodeдёӯзҡ„иЎЁзӨәгҖӮ

з”ЁPythonжөӢиҜ•дёӢпјҡ

еҸҜд»ҘзңӢеҲ°пјҢиҫ“еҮәзҡ„ж—¶еҖҷйғҪиҫ“еҮәдәҶвҖқдёӯеӣҪвҖңдёӨдёӘеӯ—пјҢеҸҜи§ҒеҶҷдёӯеӣҪе’Ң\u4e2d\u56fdзҡ„ж•ҲжһңжҳҜдёҖж ·зҡ„пјҢе№¶дё”иҜ»еҸ–зҡ„ж—¶еҖҷдёҚйңҖиҰҒеўһеҠ д»»дҪ•еҸӮж•°пјҢиҮӘеҠЁе®ҢжҲҗгҖӮ

е®һйҷ…дёҠпјҢиҝҷе№¶жІЎжңүд»Җд№ҲеҘҮжҖӘзҡ„гҖӮдёҚе°‘иҜӯиЁҖзҡ„жәҗд»Јз Ғж–Үд»¶йғҪж”ҜжҢҒеңЁеӯ—з¬ҰдёІйҮҢд»Ҙ\u4e2d\u56fdжҲ–дёӯеӣҪеҗҢж ·зҡ„ж–№ејҸиЎЁзӨәйқһASCIIеӯ—з¬ҰгҖӮ

еҸӘиҰҒи®©жӯӨж–Үжң¬зҡ„еӨ„зҗҶеҷЁжҠҠеӯ—з¬ҰдёІйҮҢзҡ„еҚ•дёӘеҸҚж–ңжқ \и§ҶдёәиҪ¬иҜ‘еӯ—з¬Ұе°ұеҸҜд»ҘгҖӮ

зңҹжӯЈйҮҚиҰҒзҡ„жҳҜдёәд»Җд№Ҳпјҹ

дёәд»Җд№Ҳпјҹ

пјҲд»ҘдёӢеҸӘжҳҜжҲ‘зҡ„дёҖдәӣзҢңжөӢпјү

еҰӮеүҚжүҖиҝ°пјҢJSONжҳҜиҰҒжұӮдҪҝз”ЁUTF-8зҡ„пјҢжүҖд»ҘжҲ‘дёҚжҳҺзҷҪдёәд»Җд№ҲиҰҒдҪҝз”ЁASCIIиҪ¬з Ғзҡ„ж–№ејҸжқҘиЎЁзӨәеҗҢж ·зҡ„еҶ…е®№гҖӮ

дҪҶиҝҷдёӘй—®йўҳе…¶е®һе№¶дёҚеңЁдәҺJSONпјҢиҖҢжҳҜеңЁдәҺж–Үжң¬ж–Үд»¶жң¬иә«пјҢиҝҳжҳҜеӯ—з¬Ұзј–з Ғзҡ„й—®йўҳгҖӮ

дёҫдёӘдҫӢеӯҗпјҢеҰӮжһңжҲ‘зҺ°еңЁжӢҝеҲ°дәҶдёҖдёӘ.jsonж–Үд»¶пјҢдҪҶжҳҜжҲ‘дҪҝз”ЁдёҖдёӘйқһеёёиҖҒзҡ„зј–иҫ‘еҷЁпјҢжҲ–иҖ…жҲ‘дҪҝз”Ёзҡ„зј–иҫ‘еҷЁдҪҝз”ЁдёҚеҗҢзҡ„зј–з Ғжү“ејҖе®ғдјҡжҖҺж ·пјҹ

еҸҜд»ҘзңӢеҲ°пјҢеҪ“дҪҝз”ЁдәҶй”ҷиҜҜзҡ„еӯ—з¬Ұзј–з Ғжү“ејҖж–Үд»¶ж—¶пјҢдҪҝз”ЁдёӯеӣҪиҝҷз§Қж–№ејҸзј–еҶҷзҡ„ж–Үд»¶еҮәзҺ°дәҶд№ұз Ғй—®йўҳпјҢиҖҢдҪҝз”Ё\u4e2d\u56fdиҝҷз§Қж–№ејҸзј–еҶҷзҡ„ж–Үд»¶еҲҷе®Ңе…ЁжІЎжңүй—®йўҳгҖӮ

е®һйҷ…дёҠиҝҷи·ҹеӯ—з¬Ұзј–з Ғзҡ„еҺҶеҸІжңүе…ігҖӮжҲ‘зӣёдҝЎжҳҜдёӘзЁӢеәҸе‘ҳйғҪзҹҘйҒ“ASCIIз ҒпјҢзӣёеҪ“дәҺеӯ—з¬Ұзј–з Ғзҡ„вҖқиҖҒзҘ–е®—вҖңдәҶгҖӮеӣ жӯӨпјҢеӨ§йғЁеҲҶзј–з ҒйғҪжҳҜи·ҹASCIIе…је®№зҡ„гҖӮдҪ еҸҜд»ҘиҜ•иҜ•зңӢпјҢеҪ“дҪ еҸӘз”ЁASCIIеӯ—з¬ҰйӣҶйҮҢзҡ„еӯ—зј–еҶҷж–Үжң¬пјҢеҪ“дҪ еҲҮжҚўзј–з Ғж—¶пјҢдҪ дјҡеҸ‘зҺ°дёқжҜ«жІЎжңүеҪұе“Қпјҡ

дҪҶжҳҜпјҢйӮЈдәӣйқһASCIIзҡ„еӯ—з¬Ұе°ұжІЎйӮЈд№ҲеҘҪиҝҗдәҶпјҢе°Ҫз®ЎзҺ°еңЁеӨ§е®¶йғҪеңЁжҸҗеҖЎз”ЁUTF-8пјҢдҪҶе…ҚдёҚдәҶиҝҳжҳҜжңүз”Ёе…¶е®ғеӯ—з¬ҰйӣҶзҡ„ж–Үд»¶гҖӮ

еӣ жӯӨпјҢеҰӮжһңдҪ дҪҝз”ЁдёҖејҖе§ӢдҪҝз”Ёзҡ„жҳҜиҪ¬иҜ‘зҡ„ж–№ејҸпјҢйӮЈд№Ҳжү“ејҖж–Үд»¶ж—¶е°ұдёҚдјҡеҮәзҺ°д№ұз ҒпјҢиҖҢз”ЁжҲ·жң¬иә«зңӢеҲ°\u4e2d\u56fdеӨ§жҰӮд№ҹиғҪзҢңеҲ°иҝҷжҳҜйңҖиҰҒиҪ¬иҜ‘зҡ„гҖӮ

иҝҷеҸӘжҳҜдёӘдҫӢеӯҗгҖӮзҺ°е®һдёӯеҸҜиғҪжңүжӣҙеӨҚжқӮзҡ„жғ…еҶөпјҢжҜ”еҰӮеҪ“ж•°жҚ®жөҒз»ҸдёҖдёӘз®ЎйҒ“ж—¶пјҢз®ЎйҒ“еҸӘиғҪеӨ„зҗҶASCIIз¬ҰпјҢеҗҰеҲҷе°ұдјҡжҠҘй”ҷпјҢиҝҷз§Қжғ…еҶөдёӢиҪ¬иҜ‘д№ҹжҜ”иҫғйҖӮз”ЁгҖӮ

йҖүе“Әз§Қпјҹ

ж—ўз„¶дҪ е·Із»ҸзҹҘйҒ“дәҶеҢәеҲ«пјҢйӮЈд№Ҳе°ұжҳҜйҖүе“Әз§Қзҡ„й—®йўҳдәҶгҖӮ


  • дҪҝз”ЁиҪ¬иҜ‘зҡ„ж–№ејҸпјҲensure_ascii=Trueпјүе…је®№жҖ§жӣҙеҘҪпјҢдёҚе®№жҳ“еҮәзҺ°д№ұз ҒпјҢдҪҶй—®йўҳе°ұжҳҜвҖқhowever those escapes will be hard to read when viewed in a text editorвҖңиҜҙзҡ„йӮЈж ·пјҢеҪ“дҪҝз”Ёж–Үжң¬зј–иҫ‘еҷЁжү“ејҖж—¶пјҢдёҚе®№жҳ“зңӢжҮӮгҖӮ

  • дҪҝз”ЁйқһиҪ¬иҜ‘зҡ„ж–№ејҸпјҲensure_ascii=Falseпјүжӣҙзӣҙи§ӮпјҢеҪ“дҪ жғіиҰҒжү“ејҖе®ғзңӢзңӢеҶ…е®№ж—¶пјҢе°ұжҜ”иҫғжҳ“дәҺзҗҶи§ЈгҖӮ


е®һйҷ…дёҠпјҢеҰӮжһңдҪ иҰҒжҺҘи§ҰеҸӘиғҪеӨ„зҗҶASCIIеӯ—з¬Ұзҡ„зі»з»ҹпјҢйӮЈе°ұжІЎеҫ—йҖүпјҢеҸӘиғҪйҮҮеҸ–иҪ¬иҜ‘ж–№ејҸгҖӮ

иҖҢжҲ‘дёӘдәәжӣҙеҒҸеҗ‘йқһиҪ¬иҜ‘зҡ„ж–№ејҸпјҢеӣ дёәзӣ®еүҚжҲ‘з”ЁеҲ°зҡ„иҪҜд»¶зі»з»ҹеҜ№UTF-8зҡ„ж”ҜжҢҒйғҪиҝҳжҳҜжҢәдёҚй”ҷзҡ„пјҢеҰӮжһңжңүиҪҜд»¶дёҚж”ҜжҢҒпјҢйӮЈжҲ‘жӣҙеҖҫеҗ‘дәҺжҚўдёҖдёӘиҪҜд»¶гҖӮ

еҰӮжһңдҪ з”ЁиҝҮIDEAпјҢдҪ дјҡзҹҘйҒ“ж–Үд»¶зј–з ҒдёӯжңүдёҖдёӘи®ҫзҪ®пјҡ

еҰӮжһңжңүе…ҙи¶Јзҡ„иҜқпјҢеҸҜд»ҘиҮӘе·ұе»әдёҖдёӘ.propertiesж–Үд»¶пјҢеңЁејҖе’Ңе…ізҡ„жғ…еҶөдёӢеҶҷдёҖдәӣйқһASCIIеӯ—з¬ҰпјҢз”Ёж–Үжң¬зј–иҫ‘еҷЁжү“ејҖзңӢзңӢгҖӮ

е»әи®®пјҡиҝҷдёӘйҖүйЎ№и·ҹдҪ зҡ„еҗҢдәӢзҡ„дҝқжҢҒдёҖиҮҙпјҢеҰӮжһңдҪ жІЎжңүеҗҢдәӢпјҢйӮЈе»әи®®дёҚеҗҜз”ЁгҖӮ
